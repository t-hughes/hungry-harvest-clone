This is a simplified version of the Hungry Harvest website.


This website is not affiliated with the real Hungry Harvest company. I wanted to create an ecommerce "clone" to showcase my skills in AngularJS, NodeJS, PostgreSQL, and MassiveJS.


The website features Angular ui-router, a simple CRUD DB made with PostgreSQL, and Bootstrap. Users can log in via facebook or local auth, choose product packages, and checkout.
